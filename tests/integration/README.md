# Integration regression data

This folder stores deterministic regression inputs for CI checks:

- `../fixtures/clip_recipes.json`: 6 tiny fixture clip recipes generated by ffmpeg during tests.
- `baseline_expectations.json`: baseline metadata constraints for probe and round-trip checks.
- `stitch_expectations.json`: seam and reconstruction thresholds for weighted tile stitching tests.

The executable tests run from `crates/helioframe-video/tests/io_regression.rs` and consume
this data to verify:

1. fixture probe metadata + frame counts,
2. decode/encode round-trip output existence,
3. baseline metadata invariants for generated outputs.

Additional stitch regression is executed by `crates/helioframe-video/tests/stitch_regression.rs`.
